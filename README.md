# 🚀 SpaceGame (Scratch)

A simple space shooter made in Scratch.  
Dodge enemies and blast them to get the highest score possible!

## 🎮 How to Play
- ⬅️ ➡️ Arrow Keys → Move ship left/right  
- ⬆️ Arrow Key → Shoot  
- Try to beat my high score of **23**!
